phyloGeoRef by dapurv5

PhyloGeoRef is a Java library for creating rich KMLs containing phylogenetic trees that can be rendered in google maps.

Input: 1) Tree file
Tree files of the following formats are supported.

  • newick
  • phyloxml
  • nexml
  • nhx
  • tol
  • nexus

2) Spatial/Meta data file
Spatial/Meta data files in the following formats are supported.
  • csv
  • txt

Output:

  • KML file
  • Compressed KMZ file

Install

Download the project. It can be opened as a netbeans project. All the necessary jars that have to be included in the classpath are placed in the dist/lib folder. Include all these in the classpath. For obtaining the latest version of these jars follow the instructions given in the README file.
